# Map

通过 `openlayer` 或者 `leaflet` 渲染地图，并实现功能交互。

本文档基于 [VitePress](https://vitepress.vuejs.org/) 构建

# 线上地址

http://map.raomaiping.host/

## 技术栈

vue3 + openlayers + leaflet

## 运行

yarn docs:dev

## 打包

yarn docs:build
